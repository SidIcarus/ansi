Category: PRINTING
Format: `CSI {p} SP e`
8-Bit Hex: `9b {p} 20 65`
7-Bit Hex: `1b 5b {p} 20 65`
Default: `{p}` = `0`
Source: [E-CF]
Title: (SCO) Select Character Orientation
Description: Rotate text.

Changes the amount of rotation for subsequent characters.

| `{p}` | Description  | Source |
|-------|--------------|--------|
| `0`   | 0 degrees.   | [E-CF] |
| `1`   | 45 degrees.  | [E-CF] |
| `2`   | 90 degrees.  | [E-CF] |
| `3`   | 135 degrees. | [E-CF] |
| `4`   | 180 degrees. | [E-CF] |
| `5`   | 225 degrees. | [E-CF] |
| `6`   | 270 degrees. | [E-CF] |
| `7`   | 315 degrees. | [E-CF] |
